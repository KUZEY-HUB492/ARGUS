🤖 ARGUS

ARGUS (Advanced Response & Guidance Utility System), Python ile geliştirilmiş hafif ve kullanışlı bir masaüstü asistanıdır.

ARGUS; uygulama açabilir, web sitelerine girebilir, sistem bilgilerini gösterebilir, kullanıcıdan yeni komutlar öğrenebilir ve birden fazla komutu aynı anda çalıştırabilir.

🌐 Web sitesi açma
🧠 Hafıza sistemi
📚 Kullanıcıdan yeni komut öğrenme
🗑️ Öğrenilen komutları unutma
📋 Öğrenilen komutları listeleme
💻 Sistem bilgisi gösterme

💬 Basit ve hızlı komut satırı arayüzü
🛠️ Kurulum
1. Depoyu klonlayın
git clone https://github.com/KUZEY-HUB492/ARGUS.git
2. Gerekli kütüphaneleri yükleyin
pip install -r requirements.txt
3. Yapılandırma

config_example.py dosyasını kopyalayıp adını config.py olarak değiştirin.

Ardından kendi bilgisayarınızdaki uygulama yollarını düzenleyin.

4. ARGUS'u başlatın
python main.py
