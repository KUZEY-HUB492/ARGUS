ARGUS (Advanced Response & Guidance Utility System), Python ile geliştirilmiş hafif ve kullanışlı bir masaüstü asistanıdır.

ARGUS; uygulama açabilir, web sitelerine girebilir, sistem bilgilerini gösterebilir, kullanıcıdan yeni komutlar öğrenebilir ve birden fazla komutu aynı anda çalıştırabilir.

✨ Özellikler 🚀 Uygulama açma 🌐 Web sitesi açma 🧠 Hafıza sistemi 📚 Kullanıcıdan yeni komut öğrenme 🗑️ Öğrenilen komutları unutma 📋 Öğrenilen komutları listeleme 💻 Sistem bilgisi gösterme 🎮 Mod sistemi (Oyun / Yayın / Geliştirme) 🔀 Aynı anda birden fazla komut çalıştırma 💬 Basit ve hızlı komut satırı arayüzü 🛠️ Kurulum

Depoyu klonlayın git clone https://github.com/KUZEY-HUB492/ARGUS.git
Gerekli kütüphaneleri yükleyin pip install -r requirements.txt
Yapılandırma
config_example.py dosyasını kopyalayıp adını config.py olarak değiştirin.

Ardından kendi bilgisayarınızdaki uygulama yollarını düzenleyin.

ARGUS'u başlatın python main.py
